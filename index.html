Instructions

Pressing the Generate button adds Amount number of random entries for Name.
Pressing the Add button adds a ticket with Number for Name if available and valid.
The Save Tickets button saves all tickets to a JSON file.
The Choose File button can load a previously saved JSON file and adds the tickets to the current ticket list.
Pressing the Pick Winner button selects a random valid number as the Winner.
REFRESHING THE PAGE CLEARS EVERYTHING
